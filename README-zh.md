# 预测市场安全分析师 (Skill)

[🇨🇳 简体中文](README-zh.md) | [🇬🇧 English](README.md)

一款先进的多语言自适应 AI Skill。专门用于为去中心化预测市场（如 Kalshi、Polymarket）提取、分析并渲染安全评估与市场逻辑护栏。

## 🌟 什么是 "Skill"？
本仓库是一个 **Skill**——一种模块化、自包含的技能包，用于扩展 AI Agent（如 Claude、Gemini 等）的能力边界。它相当于 AI 的特定领域“入职手册”，为通用大模型注入特定的专业知识、底层脚本执行力和标准工作流。

## 🚀 核心看点

- **多平台智能嗅探**：自动监听并解析用户聊天中涉及的 Polymarket 和 Kalshi 等平台的 URL 或关键字。
- **动态风险评估**：通过调用 Ansuz Security 接口，进行活体数据获取，深度评估流动性操控、智能合约漏洞和博弈规则陷阱。
- **优雅的 UI 渲染**：将干瘪的 API 接口数据（1/2/3级）转化为极度直观的、贴合用户使用语言的“安全简报”（🟢 安全 / 🟡 注意 / 🔴 危险）。
- **静默异常处理**：采用克制的错误处理机制（如在非显式提问下 API 故障时保持静默），绝对不干扰用户的主线上下文。

## 📦 如何使用

如果您的 AI 基础设施支持 `.skill` 格式，请直接将此工程导入 Agent 的工作区。

核心载荷即为根目录下的 `SKILL.md` 和 `scripts/`。导入后，AI 将自动阅读并挂载这份能力，在您探讨预测市场时默默守护。
